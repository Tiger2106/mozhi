# 五、它和VWAP的关系

很多人误解。

---

## 日线级别：

```python
amount / volume
```

本质上：

≈ 日VWAP

因为：

```text
VWAP = Σ(price × volume) / Σ(volume)
```

而：

```text
amount = Σ(price × volume)
```

所以：

```python
VWAP_day ≈ amount / volume
```

---

# 这是你现在最容易落地的VWAP

你甚至：

## 不需要逐笔数据。

日线即可。
