## 1. 判别器设计

### 特征选择原则
**全部特征与价格动量正交**，规避墨萱指出的循环论证风险：
- EXP-008/009 将验证价格动量因子在趋势市的有效性
- 若判别器本身依赖价格动量（均线/突破）分类趋势，再验证动量因子有效性 = 自我强化偏差
- 因此三个特征均从截面结构和流动性维度提取

### 特征定义

| 特征 | 计算方式 | 经济含义 |
|------|----------|----------|
| F1 截面收益离散度 | `std(pct_chg)` across A50 | 趋势市个股分化大，震荡市整体同向 |
| F2 涨跌广度比 | 上涨股数 / 50 | 持续偏向一侧反映趋势方向性 |
| F3 成交额 Z-score | 当日总成交额 vs 60日均值 | 量能放大是趋势确认的结构信号 |

### 融合规则
```
score = (F1 > 60th_pct) + (|F2 - 0.5| > 0.20) + (F3 > 0.30)

TREND   : score >= 2，且广度连续 3 日偏向同侧
RANGE   : score <= 1
NEUTRAL : 其余
```

> **无监督阈值法**，不依赖历史标注，无过拟合风险。
