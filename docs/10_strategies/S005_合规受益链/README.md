## 核心功能

识别政策合规化趋势下的受益链条，提供渐进型配置信号。

**三级受益传导链：**

```
一级：头部券商 ← 灰色资金回流 + 合规产品需求上升
  ↓
二级：低估值蓝筹 ← 散户防御性配置(3-6月) + 机构流动性改善增配(3-6月)
  ↓
三级：指数ETF ← 资金通过合规渠道流入，被动产品首当其冲
```

**渐进型建仓策略（非积极建仓）：**

- 不一次性打满，等待明确催化剂信号
- 聚焦头部3-5家券商（只有头部能承接合规资金回流）
- 低估值蓝筹分级：LDV-高股息（防御）/ LDV-慢增长（持有）/ LDV-超跌（反弹布局）

**催化剂信号库：**

| 信号 | 来源 | 操作方向 |
|:----|:----|:--------|
| QDII新额度公告 / 跨境理财通扩容 | 监管公告 | 加仓合规受益方 |
| 头部券商月报环比超+10% | 财报 | 确认受益兑现 |
| 港股通南向资金连续净流入5日 | 市场数据 | 关注AH溢价变化 |
| AH溢价指数收窄至130以下 | 市场数据 | 配置A股低估值蓝筹 |

**阈值再平衡：**
- 某标的涨幅超15% → 部分止盈
- 等待回调或下一个催化剂再补仓

## 输出形态

代码模块（Python）+ 配套文档：

- **输入**：合规渠道交易流水（交易ID、资金流向路径、渠道类型、交易金额、时间戳、参与方）
- **算法**：图遍历 → 识别受益链条（A→B→C→D路径标注受益方）
- **输出**：
  - 结构化表：chain_id, chain_path, chain_length, total_amount, terminal_beneficiary, intermediate_hubs, confidence_score
  - 统计摘要：渠道间受益传输频率、TOP-N受益链排行

## 应用环节

- 仓位管理：头寸建立节奏参考
- 信号监控：催化剂信号库触发操作建议
