## §1 背景

### 1.1 策略动机
八部门整治政策对不同企业的影响是不对称的。同一行业、同一规模的企业，因合规基础、治理结构、关联方关系的不同，在政策收紧时的受益/受损程度天差地别。识别"合规受益链"——即哪些企业会从整治政策中受益（合规优势→市场份额集中）——是投资决策的核心环节。

传统方法依赖财务报表（分析合规费用占比），但合规优势不仅体现在报表中，更体现在产业链上下游关系和关联方治理结构上。

### 1.2 要解决的问题
- 谁是合规受益者？并非所有"龙头"都会受益，核心是合规成本优势转化为竞争壁垒的企业
- 传导路径不明：政策打击灰色资金 → 行业面影响 → 头部合规企业受益，传导链长，中间环节多
- 催化剂缺失：合规受益的逻辑已经成立，但缺乏触发股价反应的"催化剂"信号
- 关联受益方遗漏：合规企业的供应商、客户、关联方可能也受益，但线性分析方法难以覆盖

### 1.3 与八部门整治政策的关系
S005 是整体策略框架的**最上层判断**——回答"整治政策来了，买什么、卖什么"。它利用 S004 的灰色路由信息识别非合规企业（灰名单），利用 S003 的政策概率判断整治强度，叠加自身的企业合规评估，生成受益链图。

## §2 设计目标

### 2.1 核心功能定位
建立三级传导链模型，识别并量化整治政策下的合规受益路径，配合催化剂信号库，输出合规受益链拓扑图和投资标的推荐。

### 2.2 预期产出

| 产出项 | 格式 | 说明 |
|:----|:----|:----|
| 合规受益链拓扑图 | JSON(图结构) | 三级传导链节点+边权重 |
| 合规评分表 | JSON | 标的企业的合规评分及排名 |
| 催化剂信号库 | JSON | 触发受益链兑现的催化剂信号列表 |
| 投资建议 | Markdown | 基于受益链的投资组合建议 |

### 2.3 使用场景
- **月频更新**：每月更新合规评分，跟踪受益链变化
- **政策冲击**：重大政策出台后，快速扫描受益链变化
- **催化剂触发**：催化剂信号出现时，触发受益链价值重估
- **S004联动**：S004新识别灰色渠道后，更新受益链中受损/受益节点

## §3 技术方案

### 3.1 三级传导链模型

**传导结构**：

```
Level 1 — 直接受益层
  合规成本优势企业（内控完善、合规投入充足的行业龙头）
    │  ↓ 与竞品对比：非合规企业退出释放市场份额
    ▼

Level 2 — 产业链协同受益层
  Level1企业的供应商、战略客户
    │  ↓ 产业联动：合规企业采购量上升 + 链上企业合规化改善
    ▼

Level 3 — 系统传导受益层
  因整治而系统性受益的行业（如金融IT改造、合规咨询、第三方审计）
    │  ↓ 行业性需求释放
    ▼

投资框架
```

**每个传导节点属性**：

| 属性 | 类型 | 说明 |
|:----|:----|:----|
| node_id | string | 企业/行业唯一标识 |
| level | int{1,2,3} | 传导层级 |
| compliance_score | float[0,100] | 企业合规评分 |
| weight | float[0,1] | 在该传导链中的影响力权重 |
| competitors_loss | float[0,1] | 竞争对手退出带来的增益系数 |
| chain_effect | float | 传导弹性（上级变化1%→本级的%变化） |

### 3.2 合规评分体系

| 维度 | 权重 | 数据源 | 说明 |
|:----|:----:|:----|:----|
| 内控合规投入 | 0.25 | 财务报告(合规费用/总营收) | 占比越高越好 |
| 治理结构评分 | 0.20 | ESG评级/董事会结构 | 独立董事比例、关联交易透明度 |
| 历史合规记录 | 0.20 | 行政处罚记录 | 近3年未被处罚加分 |
| 灰色关联度 | 0.20 | S004路由匹配 | 与灰色路由的重叠度(负向) |
| 信息透明度 | 0.15 | 信息披露质量 | 审计意见、财报及时性 |

### 3.3 图遍历算法

使用**BFS（广度优先搜索）+ 带权传导**：

```
输入：合规评分Top-10企业作为种子节点
   │
   ▼
对每个种子节点展开BFS↓↓
   ┌────────────────────────────────────────┐
   │ Level1（种子自身+直接竞争对手）          │
   │ → 计算合规优势差距（种子评分 - 竞品评分）│
   └──────────────┬─────────────────────────┘
                  ↓
   ┌────────────────────────────────────────┐
   │ Level2（产业链上下游拓展）               │
   │ → 使用供应链图谱边权重 ≤ 3跳            │
   │ → 边权重 = 交易金额/总采购额            │
   └──────────────┬─────────────────────────┘
                  ↓
   ┌────────────────────────────────────────┐
   │ Level3（行业性机会挖掘）                 │
   │ → 匹配催化剂信号库中的行业标签           │
   │ → 计算整治政策带来的行业增量需求         │
   └──────────────┬─────────────────────────┘
                  ↓
   受益链图（节点+边+权重）
```

### 3.4 催化剂信号库

催化剂信号是触发受益链价值被市场定价的事件。

| 信号类型 | 示例 | 来源 | 时效窗口 |
|:----|:----|:----|:----:|
| 合规奖励 | 某合规企业获政府表彰/补贴 | 新闻报道 | 1-2周 |
| 竞品退出 | 某非合规企业被立案调查/停业 | S004/S002 | 1-2周 |
| 行业标准 | 准入标准提高，利好合规企业 | 政策文件 | 2-4周 |
| 业绩爆表 | 合规企业季报超预期（合规优势兑现） | 财报 | 1周内 |
| 负面出清 | 行业最大违规企业被处罚，利空出尽 | S004 | 1-4周 |

信号库的**触发条件**由墨萱维护，匹配后自动输出"催化剂已触发" + 对应的受益链节点建议。

## §5 验证标准

### 5.1 通过率指标

| 指标 | 目标值 | 说明 |
|:----|:----|:----|
| Level1 识别准确率 | ≥80% | 直接受益标的与实际市场表现一致 |
| 传导链闭合率 | ≥70% | Level1→Level2→Level3的传导逻辑与实际相符 |
| 催化剂配准率 | ≥75% | 催化剂触发后相关受益链节点股价有正向反应 |
| 合规评分区分度 | ≥0.5 | Top10与Bottom10的评分差距≥50分 |
| 回溯年化超额收益 | ≥8% | 基于受益链持仓的回测超额收益 |

### 5.2 交叉验证方法
- 使用过去2年的整治事件进行回测：每件整治事件后，检验受益链是否包含事后实际受益的标的
- 每周由墨萱对Top-5推荐进行基本面审查，与S005输出比较
- 使用S002的同步指数作为控制变量，检验受益链信号是否独立于市场整体走势

## §6 风险与边界

### 6.1 已知局限性

| 风险 | 描述 | 应对 |
|:----|:----|:----|:----|
| 传导时滞 | 合规优势转化为股价可能需要数月 | 引入传导时滞参数(1-6月可调) |
| 数据可得性 | 供应链图谱并非100%完整 | 缺失节点标注"数据不可得" |
| 政策超调风险 | 整治力度过大可能让所有企业一起受损 | 增加系统性风险过滤器 |
| 合规评分主观性 | 部分维度依赖公开信息不够精准 | 标注评分置信度 |
| 催化剂漏报 | 非结构化信息难以完全覆盖 | 人工补录+周度催化剂会议 |

### 6.2 边界条件
- 合规评分数据来源任一维度连续缺数据超过2周 → 改用可用维度降维评分，标注缺失
- S004 灰色路由数据完全中断 → 合规评分"灰色关联度"维度暂不加分也不减分（设为中性）
- 传导链深度超过3层 → 停止拓展（对4层及以上的间接传导仅标注"可能受益"）

### 6.3 熔断/降级路径
1. **Level1 准确率连续2月<70%** → 回滚合规评分体系，重新校验权重
2. **催化剂频繁误报**（配准率<60%）→ 降低催化剂信号优先级，仅作参考标注
3. **系统性风险信号**（如全市场恐慌指数>90）→ 冻结受益链交易建议，输出"暂不适用"
