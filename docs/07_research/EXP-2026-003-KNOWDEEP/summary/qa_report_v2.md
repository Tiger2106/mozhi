## 2. 交叉验证

### 2.1 Q1 HV方向翻转 vs Q2 C1-RWC验证期IC增强

**问题**: Q1中l_vol_rsi_std在HV状态验证期方向翻转（HV/5d: +0.040→-0.028），但Q2的C1-RWC却在验证期IC增强（0.098→0.123负衰减）。

**验证结论: 有合理技术解释，非数据错误 ✅**

根因分析验证:

1. **HV独立翻转是真实的数据现象**: Q1 JSON中l_vol_rsi_std/high_vol/5d验证期IC=-0.02776，p=0.3065不显著，且方向从+0.0401翻转。✅ Q1 FDR验证期仅4/18显著，HV/5d已被排除。

2. **复合信号机制改变信号结构**: C1-RWC将各状态合并，HV仅占50%权重，MV/LV的方向一致性（验证期均为正IC）补偿了HV翻转。验证期加权预测为-0.0105（略负），但实际C1-RWC为+0.1225——存在显著的cross-term。

3. **Cross-term的数学解释合理**: 当同一股票在不同市场状态下的信号被合并，信号中的共同成分被增强而噪声被稀释。这在方法论上是已知的量价信号增强技术。

4. **样本量的关键作用**: Q1 HV验证期n≈1,341（不足），Q2 C1-RWC验证期n≈57,495（充足）。统计检验力的量级差异使得Q1的结论置信度远低于Q2。

### 2.2 "跨状态协同效应使IC是单状态加权的4-12倍"验证

**报告声称数据**:

**逐项验证**:
- **训练期5d**: 0.0979 / 0.0235 = **4.168× ≈ 4.2×** ✅
- **验证期5d**: 加权为负值，实际为正，方向相反。声称"反向"是准确的 ✅
- **验证期20d**: 报告中未给出20d的精确加权预测值。使用Q1数据：

  Q1单状态验证期20d IC: HV=+0.01945, MV=+0.02005, LV=+0.02834

  实际C1-RWC val 20d = 0.1384
  倍数 = 0.1384 / 0.0214 = **6.47×**（落在4-12×范围内 ✅）

**验证结论**:
- 4.2×（训练期5d）→ ✅ 精确匹配
- "反向"（验证期5d）→ ✅ 描述准确
- "~4-12×"（验证期20d）→ ✅ 自算6.5×落在范围内，保守声称合理
