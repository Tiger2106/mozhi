## 核心功能

监控"堵已上路疏在路上"的非对称风险，量化资金围栏效应。

**主指标——堵疏同步指数（墨衡）：**

```
同步指数 = 已落地的疏导性措施数量 × 权重 / 已落地的限制性措施数量 × 权重
```

| 指数范围 | 判定 | 资金去向概率分布 |
|:--------|:----|:---------------|
| ≥0.8 | 堵疏同步 | 40% A股 / 30% 债市 / 20% 港股通 / 10% 闲置 |
| 0.4-0.8 | 堵快疏慢 | 20% A股 / 35% 债市 / 15% 港股通 / 20% 另类 / 10% 闲置 |
| <0.4 | 严重失衡 | 10% A股 / 25% 债市 / 35% 另类 / 30% 闲置 |

**辅指标——错配热力图（墨萱）：**

```
错配值 = f(堵的力度) − g(疏的进度)
```
- 覆盖渠道：A股/债市/另类资产
- 输出：各渠道资金压力指数0-100，绿/黄/橙/红四级预警
- 连续3周上升 → 主动推送预警

**参照系——历史案例库（玄知）：**

- V1（2026年5月）：近3年（2023-2025）重大资金跨境/反洗钱案例10-15个
- V2（V1后2周）：扩充至近5年，补充大湾区/自贸区试点案例20-25个
- V3（视需）：FATF框架下国际对标案例5-8个
- 更新频率：常规2周一次，重大政策发文24h内入库

## 三条线格式统一约束（墨衡协调人）

| 维度 | 要求 |
|:----|:-----|
| 时间轴 | 统一日频，按交易日对齐 |
| 标的池 | A50核心池 + 拓展池两级 |
| 数据粒度 | 标的-日期-渠道三维 |
| 输出元组 | {date, symbol, channel_type, value} |

**墨衡输出格式：** `synchronicity_index_{date}.json` → {date, symbol, block_score, dredge_score, synch_ratio, trend_signal}

**墨萱输出格式：** `mismatch_heatmap_{date}.json` → {date, symbol, channel_pair, mismatch_intensity, z_score}，channel_pair取值与墨衡channel_type枚举一致

**玄知输出格式：** `historical_reference_{case_id}.json` → {case_id, time_range_start, time_range_end, symbols[], pattern_match_score, notes}，归一化方法一致

## 里程碑节点检查

每季度检查疏导进度（合规产品备案数量、跨境额度批出量、细则出台率），里程碑滞后超1个季度 → 提升风险等级。
