## 核心功能

将灰色资金回流规模从单点估算升级为结构化情景推演，含折扣因子链推导、三情景输出、宏观修正和偏差归因闭环。

**四重折扣因子链（共识参数）：**

```
理论上限 → [港股通分流 0.65] → [合规瓶颈 0.78]
         → [执行折扣 0.85] → [债市分流 0.85] → 实际回流估算
```

综合折扣率：0.65 × 0.78 × 0.85 × 0.85 ≈ **0.366**

**三情景输出：**

| 情景 | 回流规模 | 概率权重 | 条件 |
|:----|:--------|:--------:|:-----|
| 乐观 | 3500-5000亿 | 25% | 各折扣取上限，基准成交1.2万亿 |
| 中性 | 2000-3500亿 | 50% | 各折扣取中值，基准成交1.0万亿 |
| 悲观 | 800-1500亿 | 25% | 各折扣取下限，基准成交0.8万亿 |

## 数据接口（墨衡→墨萱）

**输入字段（墨衡输出）：**

| 字段 | 类型 | 说明 |
|------|------|------|
| scenario_id | string | 情景标识 |
| scenario_name | string | 情景名称 |
| date | date | 时间戳 |
| expected_inflow | float | 预期回流资金（亿元/日） |
| actual_inflow | float | 实际回流资金（亿元/日） |
| deviation | float | actual - expected |
| deviation_pct | float | (actual-expected)/expected×100% |
| confidence_band_lower | float | 情景置信区间下限 |
| confidence_band_upper | float | 情景置信区间上限 |
| key_factors[] | array | 墨衡已识别驱动因子列表 |

**输出字段（墨萱补充偏差归因后）：**
预留字段：deviation_bucket + attribution_reason

**接口协议：** CSV或JSON，墨衡完成每版情景后群内通知"情景引擎vX.Y已发布"，墨萱自行拉取。不并行。
