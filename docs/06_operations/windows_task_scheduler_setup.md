# Windows 任务计划配置指南

**作者:** 墨衡
**更新时间:** 2026-05-16

## 目的

配置 Windows 任务计划程序，使系统每天凌晨 02:00 自动执行 `scripts/daily_maintenance.bat` 脚本，完成墨枢平台的每日例行运维任务。

## 适用环境

- Windows 10 / Windows 11（专业版/企业版/工作站版）
- 需管理员权限

## 步骤

### 1. 打开任务计划程序

按 `Win + R` 打开运行对话框，输入 `taskschd.msc`，回车：

```
taskschd.msc
```

![Win+R 运行 taskschd.msc](images/placeholder_run_taskschd.png)

> 如果提示"无权访问"，请以管理员身份运行（右键开始菜单 → Windows 工具 → 任务计划程序）。

### 2. 创建基本任务

1. 在右侧操作栏中，点击 **创建基本任务…**
2. 按如下填写向导：

| 页面 | 设置 |
|------|------|
| **名称** | `墨枢每日运维` |
| **描述** | `每天 02:00 执行 daily_maintenance.bat` |
| **触发器** | **每天** → 开始时间：`02:00:00` |
| **操作** | **启动程序** → 程序或脚本：浏览选择 `scripts/daily_maintenance.bat` |

![创建基本任务向导](images/placeholder_basic_task_wizard.png)

> ⚠️ `.bat` 路径建议使用绝对路径，例如：`C:\Users\17699\mozhi_platform\scripts\daily_maintenance.bat`。

### 3. 修改任务属性（关键步骤）

创建完成后，在任务列表中找到 **墨枢每日运维**，**右键 → 属性**，修改以下配置：

#### 3.1 常规选项卡

- ✅ **不管用户是否登录都要运行**
- ✅ **使用最高权限运行**
- ✅ 配置适用于：**Windows 10 / Windows Server 2016**（或你的当前系统版本）

![常规选项卡设置](images/placeholder_general_tab.png)

#### 3.2 条件选项卡

**取消勾选**以下两项（确保任务在笔记本电脑接电前也能在指定时间触发）：

- ❌ ~~只有在交流电时才启动~~
- ❌ ~~只有在网络连接时才启动~~

![条件选项卡设置](images/placeholder_conditions_tab.png)

其余配置保持默认即可。

#### 3.3 设置选项卡（可选确认）

建议确认以下选项已勾选（通常为默认值）：

- ✅ **允许按需运行任务**
- ✅ **如果任务运行时间超过以下时间，停止任务** → 设为 `1 小时`

![设置选项卡](images/placeholder_settings_tab.png)

### 4. 确认配置

回到任务计划程序主界面，在任务列表中应看到：

| 字段 | 预期值 |
|------|--------|
| 任务名 | 墨枢每日运维 |
| 触发器 | 每天 02:00 — 触发一次 |
| 状态 | 就绪 |
| 下次运行时间 | 次日 02:00 |

## 验证

### 1. 立即运行测试

在任务列表中右键 **墨枢每日运维** → **运行**。

### 2. 确认首次输出

查看任务运行状态和历史：

1. 选中该任务 → 点击下方 **显示所有任务运行历史记录**
2. 确认最近一条记录为：**操作已完成**（任务完成代码 0x0）
3. 检查 `scripts/daily_maintenance.log`（若脚本有日志输出）是否有内容

![运行历史记录](images/placeholder_run_history.png)

### 3. 常见问题排查

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 任务不执行 | 未勾选"不管用户是否登录" | 回到常规选项卡勾选 |
| 运行时权限不足 | 未勾选"最高权限" | 回到常规选项卡勾选 |
| 笔记本合盖不执行 | 未取消"交流电条件" | 回到条件选项卡取消勾选 |
| 找不到 .bat 路径 | 使用了相对路径 | 改为绝对路径 |

## 附：修改或删除任务

- **修改任务**：在任务列表中双击任务名，直接编辑各选项卡
- **禁用任务**：右键 → 禁用
- **删除任务**：右键 → 删除（此操作不可撤销）

---

*配置完成后，墨枢平台将在每天凌晨 02:00 自动执行运维脚本，无需手动干预。*
