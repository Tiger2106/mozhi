## 2. 深度逻辑推演

### 2.1 宏观驱动因素

三个因子在 high_vol/20d 组合下同时通过 FDR 校正，存在以下合理解释：

**2.1.1 l_vol_rsi_std (高波动 RSI 波动率)**
- 在高波动市场状态下，RSI 的离散度（std）反映了个股间的短期超买/超卖分化程度
- 20天持有期足够让均值回归效应显现，高离散 → 后续强势股分化 → IC 为正
- 逻辑链条完整：高波动 → RSI 分散 → 回归动能 → 区分度增强

**2.1.2 TrendQuality (趋势质量)**
- 原始负向 IC（-0.0288）说明高趋势质量的股票在高波动期后续表现反而偏弱
- 反转逻辑：高波动期市场交易过度，趋势质量高的标的（已充分定价）后继乏力，低趋势质量标的反而有反弹空间
- 逻辑自洽：反转效应在波动率放大时最为明显

**2.1.3 l_str_kdj_k (KDJ K 值动量)**
- 原始负向 IC（-0.0192），反转后正向
- KDJ K 值反映了短期超买/超卖状态；高波动期超买后回调压力大于超卖后反弹
- 但高波动期下超卖反弹效应与超买回调效应不对称，总体呈现轻微负向

### 2.2 资金可持续性

| 维度 | 评估 |
|:----:|:----:|
| IC 稳定性 | l_vol_rsi_std 的 high_vol 状态下 4 个持有期全部通过 L3 稳定性，滚动翻转率 0.0% |
| OOS 一致性 | l_vol_rsi_std high_vol/20d 的 IS IC=+0.0436 vs OOS IC=+0.0367（方向一致，衰减 16%），可接受 |
| 参数稳健性 | 9 个阈值网格点 243 组合 0 翻转，参数完全不敏感 |
| 状态依赖性 | 仅在 high_vol 状态下有效；low/mid_vol 无足够数据验证 |

**结论**：因子在 high_vol 状态下具备中等可持续性，l_vol_rsi_std 可持续性最优。

### 2.3 催化剂 vs 趋势

当前回测为因子 IC 检验，非事件驱动策略，不存在催化剂概念。但可类比：
- **TrendQuality** 在 high_vol/20d 的负向信号 = 反转效应（均值回归），而非趋势延续
- **l_vol_rsi_std** 的正向信号 = 分化效应（高 RSI 离散 → 截面区分度）
- **l_str_kdj_k** 的微弱负向信号 = 超买回调效应

三者均属于**结构性市场微观结构效应**，而非宏观趋势性催化。

## 7. FDR BH 校正深度核查

### 7.1 校正流程验证

```

 0.0000 → 0.0001 → 0.0086 → 0.0316 → 0.0479 → 0.0578
→ 0.0628 → 0.0702 → 0.1789 → 0.1856 → 0.2027 → 0.3754

BH 阈值：q × (i/m) = 0.05 × i/12

i=1:  0.05×1/12 = 0.00417 → 0.0000 < 0.00417 ✅ 拒绝
i=2:  0.05×2/12 = 0.00833 → 0.0001 < 0.00833 ✅ 拒绝
i=3:  0.05×3/12 = 0.01250 → 0.0086 < 0.01250 ✅ 拒绝
i=4:  0.05×4/12 = 0.01667 → 0.0316 > 0.01667 ❌ 停止
```

**验证结论**：与前 3 个 p 值被拒绝正确匹配，实现逻辑无误。
