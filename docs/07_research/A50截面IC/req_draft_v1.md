## 1. 项目背景与目标

### 1.1 背景

当前因子研究管线依赖时间序列IC（`compute_factor_ic.py`），仅评估单个因子在时间序列上的预测能力，无法回答"某个因子在全体成分股截面上是否具有选股区分度"这一核心问题。截面IC是因子有效性验证的行业标准方法，是多因子模型构建的基础前提。

### 1.2 数据现状（墨涵调查确认）

| 项目 | 现状 |
|:----|:-----|
| 数据源 | `market_data.db.stock_daily` — tushare_pro |
| 覆盖范围 | 上证50成分股日线行情 |
| 时间跨度 | 2007-01-04 至 2026-05-26 |
| 字段数量 | 26个（含 ts_code, trade_date, open/high/low/close/pre_close, volume, amount, turnover_rate, pe, pb, adj_factor, total_mv, circ_mv, free_float 等） |
| 截面IC管线 | ❌ 不存在 |
| 成分股清单表 | ❌ 不存在（无正式 `a50_universe` 表） |
| 现有IC计算 | 时间序列IC，非截面IC |

### 1.3 目标

1. 建立上证50全成分股的截面IC计算管线
2. 构建独立IC结果存储表，支持多因子同步测试与回溯
3. 以单因子"黄金基线"作为管线验证硬性前置条件
4. 实现30天内可交付的最小可行产品（MVP）

## 2. 范围与边界

### 2.1 包含

- **标的范围**：上证50全部成分股（历史成分股 + 现时成分股）
- **因子范围**：10-15个候选因子，分五大类别（动量、反转、质量、估值、波动/流动性）
- **计算频率**：周频（每周末截面快照）
- **时间范围**：2007-01-04 至 2026-05-26
- **IC计算类型**：
  - 普通截面IC（Pearson相关系数）
  - 秩相关IC（Spearman相关系数）
  - p-value 显著性检验
  - 调整IC（剔除极端值影响）
- **输出指标**：IC值、Rank IC、p-value、有效样本数、调整IC
- **交付物**：
  - 数据摸底报告（前置条件）
  - Schema冻结文档
  - 数据落库 + ETL脚本
  - 截面IC计算管线
  - 单因子黄金基线验证报告
  - 多因子初步分析报告

### 2.2 不包含

- ❌ 不构建多因子组合模型（此为后续迭代）
- ❌ 不处理非A50成分股（标的范围严格限定）
- ❌ 不处理分钟级/日频以下数据
- ❌ 不包含机器学习/深度学习因子挖掘
- ❌ 不包含实时计算或盘中更新（仅离线批处理）
- ❌ 不包含IC衰减分析和换手率优化（后续迭代）

## 3. 数据层需求

### 3.1 源数据表：`a50_daily_ohlcv`

从 `market_data.db.stock_daily` 提取上证50成分股数据，构建独立视图/物化表。

#### 3.1.1 字段定义

| 字段名 | 类型 | 约束 | 描述 | NULL语义 |
|:------|:----|:-----|:----|:---------|
| `id` | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 | 无NULL |
| `ts_code` | TEXT | NOT NULL | 股票代码（如 600000.SH） | 无NULL |
| `trade_date` | TEXT | NOT NULL | 交易日（YYYYMMDD） | 无NULL |
| `open` | REAL | | 开盘价（复权后统一口径） | 停牌日=NULL |
| `high` | REAL | | 最高价 | 停牌日=NULL |
| `low` | REAL | | 最低价 | 停牌日=NULL |
| `close` | REAL | NOT NULL | 收盘价（复权后统一口径） | **缺失≠停牌**：需区分NULL语义（见备注） |
| **`pre_close`** | REAL | | **前一交易日收盘价**（墨萱要求补充） | 复权更新后的前一交易日价 |
| `volume` | REAL | | 成交量（股） | 停牌日=0 |
| `amount` | REAL | | 成交金额（元） | 停牌日=0 |
| `turnover_rate` | REAL | | 换手率（%） | 缺失=NULL |
| `pe` | REAL | | 市盈率 | 缺失=NULL |
| `pb` | REAL | | 市净率 | 缺失=NULL |
| `adj_factor` | REAL | NOT NULL | 当日复权因子 | **必须非NULL**；停牌日沿用前一交易日值 |
| `total_mv` | REAL | | 总市值 | 缺失=NULL |
| `circ_mv` | REAL | | 流通市值 | 缺失=NULL |
| `free_float` | REAL | | 自由流通股本比例 | 缺失=NULL |
| `source_version` | TEXT | NOT NULL, DEFAULT 'v1' | 数据版本锚定（玄知要求） | 无NULL，默认'v1' |
| `created_at` | TEXT | NOT NULL | 记录创建时间（ISO8601+08:00） | 无NULL |

> **NULL语义约定（墨萱T+7验收要求）**：
> - **NULL-缺失**：API未返回或源表无记录 → 标记 `null_reason='MISSING'`
> - **NULL-停牌**：当日停牌导致价格字段不可用 → 标记 `null_reason='SUSPENDED'`
> - 建议增加 `null_reason` 字段（TEXT, 可为NULL）区分以上情况
> - 精度约束：价格字段统一 `REAL`，精度不低于小数点后4位；金额字段精度不低于小数点后2位

#### 3.1.2 索引

```sql
CREATE UNIQUE INDEX idx_a50_daily_pk ON a50_daily_ohlcv(ts_code, trade_date);
CREATE INDEX idx_a50_daily_date ON a50_daily_ohlcv(trade_date);
CREATE INDEX idx_a50_daily_code ON a50_daily_ohlcv(ts_code);
```

#### 3.1.3 复权方案选择（玄知+墨萱重点关注）

| 方案 | 说明 | 推荐度 |
|:----|:----|:------:|
| **后复权** | 使用 `adj_factor` 将价格统一向后复权 | ⭐ **推荐（默认）** |
| 前复权 | 以最新交易日为基准向前复权 | 可选（需统一口径） |
| 不复权 | 使用原始价格 | 不推荐（截面口径不可比） |

**决策依据**：截面IC计算的核心要求是**所有成分股在同一时间截面上的价格口径一致**。后复权使用 `close * adj_factor` 即可得到统一可比价格，且不随最新交易日变动，便于历史回溯。

> **复权一致性校验**（墨萱重点关注）：
> - 写入后必须检查：`adj_factor[t] / adj_factor[t-1] ≈ close[t] / pre_close[t]`（前复权约束）
> - 如口径偏了全表不可回溯 → 需在ETL中加入自动校验

## 4. 计算层需求

### 4.1 因子定义（多因子同步测试，分五大类）

#### 4.1.1 动量类（Momentum）

| 因子名 | 说明 | 计算方式 | 预期方向 |
|:------|:----|:--------|:--------:|
| `momentum_5d` | 5日动量 | `(close[t]/close[t-5]) - 1` | 正 |
| `momentum_20d` | 20日动量（月） | `(close[t]/close[t-20]) - 1` | 正 |
| `momentum_60d` | 60日动量（季） | `(close[t]/close[t-60]) - 1` | 正 |
| `momentum_120d` | 120日动量（半年） | `(close[t]/close[t-120]) - 1` | 正 |

#### 4.1.2 反转类（Reversal）

| 因子名 | 说明 | 计算方式 | 预期方向 |
|:------|:----|:--------|:--------:|
| `reversal_1d` | 日内反转 | `open[t]/close[t-1] - close[t]/open[t]` | 负 |
| `reversal_5d` | 5日反转 | `-(close[t]/close[t-5] - 1)` | 负 |

#### 4.1.3 质量类（Quality）

| 因子名 | 说明 | 计算方式 | 预期方向 |
|:------|:----|:--------|:--------:|
| `roe_ttm` | ROE（TTM） | 净资产收益率（从财报数据推导或使用 `pe * eps` 反算） | 正 |
| `profit_margin` | 毛利率 | 营业收入利润率（需补充财报数据） | 正 |

#### 4.1.4 估值类（Valuation）

| 因子名 | 说明 | 计算方式 | 预期方向 |
|:------|:----|:--------|:--------:|
| `pe_ttm` | 市盈率TTM | 源表 `pe` | 负 |
| `pb_lf` | 市净率 | 源表 `pb` | 负 |
| `dividend_yield` | 股息率 | 近12月每股分红 / 价格（需补充分红数据） | 正 |

#### 4.1.5 波动与流动性类（Volatility / Liquidity）

| 因子名 | 说明 | 计算方式 | 预期方向 |
|:------|:----|:--------|:--------:|
| `volatility_20d` | 20日波动率 | 过去20日收益率的标准差 | 负 |
| `turnover_20d_avg` | 20日均换手率 | 过去20日换手率均值 | 负 |
| `illiquidity_20d` | Amihud非流动性指标 | 20日均 `\|return\| / volume` | 负 |
| `volume_20d_change` | 20日成交量变化率 | `volume[t]/avg(volume[t-5:t]) - 1` | 正 |

**总计：15个候选因子，5大类别**（墨衡推荐：10-15个，分五类）

### 4.2 频率：周频

- **截面日期选择**：每**周五**（若周五非交易日则选周四，以此类推）
- **回看窗口**：因子计算使用至截面日为止的完整历史数据
- **预测窗口**：截面IC计算 `rank(factor[t])` vs `return[t+1:t+5]`（未来一周收益）
- **滚动方式**：每周末滚动快照，不重叠

### 4.3 单因子黄金基线（T+21硬性前置条件）

#### 4.3.1 选择标准

选择**最简单、最稳健、行业公认有效**的单因子作为管线验证基线：

| 基线因子 | 说明 | 选为基线的理由 |
|:---------|:----|:--------------|
| **`momentum_20d`** | 过去20日（约1个月）动量 | 1. 计算简单：仅需收盘价<br>2. 行业标准：Fama-French体系中基础因子<br>3. 可复现：几乎所有学术论文和量化社区有参考值<br>4. 数据稳定：不需要额外财报数据 |

#### 4.3.2 基线验证指标

| 指标 | 期望范围 | 验证方式 |
|:----|:--------|:---------|
| 均值IC | > 0 (正预期) | 全样本计算后取均值 |
| IC正比率 | > 50% | 全体IC中正值的占比 |
| Rank IC稳定性 | IC标准差 < 0.2 | 衡量IC变异程度 |
| p-value 显著比率 | < 0.05 的占比 > 30% | 统计显著的比例 |
| IC半衰期 | > 12周 | IC序列自相关半衰期 |

#### 4.3.3 失败标准与修复路径

| 失败情形 | 可能原因 | 修复动作 |
|:--------|:--------|:--------|
| 均值IC < 0 | 复权口径不一致、涨跌停未处理 | 检查复权逻辑、处理极端值 |
| 有效样本数不足 | 停牌日处理不当 | 修正停牌识别逻辑 |
| IC大幅偏离文献值 | 周频截面定义偏差（日期选择错误） | 验证周频截面日期选择逻辑 |
