## ═══════════════════════ 3. 边界情况测试 ═══════════════════════

| 测试场景 | 触发方式 | 结果 | 说明 |
|:---------|:---------|:----|:-----|
| volume=0 | `np.where(vol>0, amt/vol, NaN)` | ✅ **返回 NaN** | `np.where` 条件保护，不执行零除 |
| volume=NaN | 同上 | ✅ **返回 NaN** | NaN > 0 = False → NaN |
| amount=0 | 同上 | ✅ **返回 0.0** | 0 / vol = 0 |
| 极端大值（1e12/1e10） | - | ✅ **返回 100.0，无溢出** | float64 安全 |
| 真实零交易量日 | 全库扫描 | ✅ **0 个记录** | 所有交易日均有正常成交 |

**保护逻辑验证：**
```python
result = np.where(
    df['volume'].values > 0,           # 条件保护
    df['amount'].values / df['volume'].values,  # 仅当 vol > 0 时取值
    np.nan                             # vol <= 0 时返回 NaN
)
```
即使 numpy 会预先计算除法，`np.where` 的条件路由会确保 vol=0 时不取除法结果。

## ═══════════════════════ 4. 与 VWAP 因子对比分析 ═══════════════════════

### 4.1 结构性差异

| 对比维度 | 现有 VWAP（vwap_factor.py） | **成交额均价（amount/volume）** |
|:---------|:--------------------------:|:------------------------------:|
| 数据源 | `typical_price = (high+low+close)/3` × `volume`（估算） | `amount` = 交易所统计的实际成交总额 |
| 计算方式 | 滚动窗口累计（cumsum / rolling） | 单日除法 |
| 精度 | **近似值** — bar 级估算 | **精确值** — 交易所汇总值 |
| 窗口支持 | 5/10/20 日等 | 可扩展（raw/rolling/ema） |
| 信号含义 | 机构持仓成本线（长期趋势） | 当日实际成交均价（短期） |

### 4.3 统计对比

| 统计量 | **avg_price** | VWAP(cum) |
|:------|:------------:|:---------:|
| **mean** | 13.82 | 12.89 |
| **std** | 3.66 | 0.95 |
| **相关性** | **0.49**（000001.SZ）/ **0.91**（601857.SH） | - |

### 4.4 关键结论

1. **不是冗余，是互补** — avg_price 的日内灵敏度远高于 VWAP，VWAP 是累积平滑值
2. **偏差本身是信号** — `avg_price - VWAP` 的偏离可以指示当日成交结构（买方 vs 卖方主导）
3. **换一种方式看**：VWAP 回答"机构重仓成本"，avg_price 回答"今天大家实际成交均价"

## ═══════════════════════ 5. 因子实现验证 ═══════════════════════

### 5.1 AvgTradePriceFactor 类

已完成试验版实现（走 `BaseFactor` 继承标准路径）：

| 属性 | 值 |
|:-----|:----|
| 类名 | `AvgTradePriceFactor(BaseFactor)` |
| 名称 | `avg_trade_price` |
| 分类 | `volume`（量能因子族） |
| 默认参数 | `{rolling_window: 1, smoothing: "raw"}` |
| 输入列 | `amount`, `volume` |
| 保护逻辑 | `np.where(vol > 0, amt/vol, NaN)` |

支持参数：
- `rolling_window`：滚动均值窗口（默认1=单日）
- `smoothing`：平滑方式（`raw` 原始值 / `ema` 指数平滑）

### 5.2 FactorRegistry 注册测试

| 测试 | 结果 |
|:-----|:-----|
| 注册 `registry.register_factor("avg_trade_price", ..., "volume")` | ✅ 成功 |
| 列表中出现 `registry.list_factors('volume')` | ✅ 包含 |
| `registry.compute_all(df)` 返回数值 | ✅ 正常 |
| 注销 `registry.unregister("avg_trade_price")` | ✅ 成功 |

### 5.3 与指南 §2.2 / §3.2 对齐

因子列表（volume 族）需增加条目：

```yaml
avg_trade_price:
  name: 成交额均价
  formula: amount / volume
  category: volume
  description: 当日实际成交均价（交易所统计的成交额/成交量）
  inputs: [amount, volume]
  params:
    rolling_window: 1
    smoothing: raw | ema
  relationships:
    complement_to: vwap (VWAP 是累积估算值，avg_trade_price 是单日精确值)
```
