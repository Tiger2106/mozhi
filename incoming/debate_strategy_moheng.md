## 策略一：灰色资金链路由追踪系统

### 策略名称
灰色资金链路由追踪系统

### 具体内容
1. **两层拆分监测**：将资金链拆分为"供给端"和"需求端"分别建模。供给端关注灰色通道的关闭速率（刑事追诉案件数、银行账户冻结量级），需求端关注套利空间（AH溢价、境内融资成本与境外配资利差）。
2. **渠道变形预警**：辩论揭示灰色渠道在打击下"不会消失只会变形"。系统应建立渠道变形跟踪列表（场外TRS→信用贷置换→虚拟货币跨境→虚假贸易购汇等），在旧渠道受阻时自动标记新变体的出现概率。
3. **合规/风险分层**：将TRS配资链条分为三层分别打标：
   - 纯合规层（持牌机构直营）→ 安全，不加仓不减仓
   - 合规包装层（合规产品经灰色中介转售+多层嵌套）→ 高风险，触发减持信号
   - 纯灰色层（非持牌TRS）→ 极高风险，触发立即预警

### 应用场景
- **Step2 深度分析**：注入 datacollection 中 scope 字段的资金链路追踪数据
- **Step4 质量审查**：验证报告中对灰色资金风险的评估是否覆盖了上述三层结构
- **XBOT 风险模块**：作为资金链路异常检测的规则引擎输入

### 触发条件
- 八部门/多部门联合整治文件发布 → 立即启动供给端监控
- AH溢价率单日变动超过历史2σ → 检查是否存在灰色渠道变形
- 任一灰色渠道传出大规模冻结/追诉消息 → 触发渠道变形扫描

## 策略二：政策"堵疏"同步性监测指标

### 策略名称
"堵疏同步性"监测与资金围栏效应评估

### 具体内容
辩论明确揭示了一个非对称风险：**"堵已上路，疏在路上"**——限制性政策快速落地，而疏导性配套滞后，形成资金围栏效应。

1. **建立堵疏同步指数**：
   - 分子 = 已落地的疏导性政策条款数量 × 权重（如QDII扩额、港股通优化、合规跨境产品审批）
   - 分母 = 已落地的限制性政策条款数量 × 权重（如灰色TRS查处、净卖空限制、账户冻结）
   - 指数 < 0.6 → 触发围栏效应预警（资金被堵但无出路，风险向债市/另类资产/流动陷阱传导）

2. **量化不同场景下的资金去向概率分布**（辩论第二轮的延伸）：
   - 堵疏同步（指数≥0.8）：A股流入40% / 债市30% / 港股通20% / 闲置10%
   - 堵快疏慢（指数0.4-0.8）：A股流入20% / 债市35% / 港股通15% / 另类资产20% / 闲置10%
   - 严重失衡（指数<0.4）：A股流入10% / 债市25% / 另类资产35% / 闲置30%

3. **设置2年过渡期里程碑节点检查**：
   - 每季度检查一次疏导进度（合规产品备案数量、跨境额度批出量、政策细则出台率）
   - 里程碑滞后超过1个季度 → 提升风险等级

### 应用场景
- **Step2 深度分析**：将堵疏指数作为 risk_assessment 的定量输入，替代纯定性的风险描述
- **Step3 报告生成**：自动在报告中插入"堵疏同步性"进度条可视化
- **Step5 决策建议**：堵疏指数影响操作建议激进程度（指数低则强制偏向 conservative 档）

### 触发条件
- 任何一轮多部门联合整治政策发布后 → 初始化堵疏指数基线
- 每季度初 → 检查疏导里程碑进度
- 堵疏指数连续两个月低于0.5 → 输出"围栏效应"专题分析报告

## 策略三：资金回流规模情景化推演框架

### 策略名称
资金回流规模情景化推演框架

### 具体内容
辩论中三方对资金回流量级有显著分歧（乐观3500-5000亿 / 中性2000-3500亿 / 悲观800-1500亿），分歧背后是多重折扣因子的叠加效应。

1. **建立四重折扣因子链**（从辩论第三轮提炼）：
   ```
   理论回流上限 → [港股通分流折扣] → [合规瓶颈折扣]
                   → [执行折扣] → [债市分流折扣] → 实际回流估算
   ```
   - 各折扣因子设可调参数，默认值从辩论共识确定：
     - 港股通分流：0.60-0.70（中值0.65）
     - 合规瓶颈：0.70-0.85（中值0.78）
     - 执行折扣：0.70-0.95（辩论修正后中值0.85）
     - 债市分流：0.80-0.90（中值0.85）

2. **三情景引擎**：
   - 乐观情景：各折扣取上限 + 基准成交1.2万亿 + 无宏观冲击
   - 中性情景：各折扣取中值 + 基准成交1.0万亿 + 正常宏观
   - 悲观情景：各折扣取下限 + 基准成交0.8万亿 + 宏观下行叠加
   - 输出：点估计 + 区间 + 概率权重（参考辩论修正后的执行概率分布：雷霆30%/运动式35%/选择性25%/虎头蛇尾10%）

3. **接入宏观环境修正因子**：
   - 辩论指出基准回调风险（4月零售+0.2%说明宏观下行正在发生）
   - 若宏观指标（零售/PMI/社融）连续两个月低于预期 → 下调基准成交假设，同时收紧所有折扣因子

4. **设置6个月验证锚点**：
   - 3个月验证点（2026年8月）：检查实际成交增量 vs 预测区间
   - 6个月验证点（2026年11月）：检查券商受益兑现 + AH溢价变化
   - 验证偏差超过30% → 重新校准折扣因子参数

### 应用场景
- **Step2 深度分析**：替代当前对回流的单点估算，改用三情景推演
- **XBOT 信号生成**：三情景的概率权重影响 signal_mapping 的 confidence 和 position_ratio
- **Step4 质量审查**：检查报告中对回流规模的估算是否使用了折扣因子链结构，避免孤立的乐观/悲观单点结论
- **运营仪表盘**：在墨枢决策看板中展示三情景实时更新

### 触发条件
- 整治政策发布后24小时内 → 初始化三情景框架，输出基线推演
- 月度宏观经济数据公布 → 更新基准成交假设
- 季度验证点 → 对比实际数据与预测，决定是否重新校准折扣因子
- 任何新增汇率/利率/监管信息 → 触发单一折扣因子微调
