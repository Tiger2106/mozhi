> 提取人：墨萱（第三方测试员）
> 提取日期：2026-05-25

## 策略一：灰色资金回流情景推演与验证

**策略名称**：灰色资金回流多情景验证

**具体内容**：

1. **设定三情景基准**：
   - 乐观情景：回流 3500-5000 亿
   - 中性情景：回流 2000-3500 亿
   - 悲观情景：回流 800-1500 亿（考虑港股通分流 + 合规瓶颈 + 执行折扣 + 债市分流的四重折扣）
   - 基准假设：全市场日均成交 1 万亿

2. **结构化追踪回流路径**：
   - 通道：灰色 TRS → 回购渠道 → 直接入市
   - 分流：港股通 / 债市 / 另类资产
   - 瓶颈：合规产品备案周期、私募发行节奏

3. **定期校准机制**：
   - 每月对比场景回流的实际资金规模，判断当前处于哪个情景
   - 若连续 2 个月偏离该情景，自动切换更接近的实际情景

4. **偏差归因**：
   - 记录实际值与预测值之差，归因到四重折扣因素中哪一项超预期

**应用场景**：墨枢的「宏观分析模块·政策影响评估子模块」
- 在政策分析流程中，八部门整治这类事件触发后，自动执行情景设定与追踪
- 在每周/每月市场监控环节调用，作为资金面向下最核心的验证指标

**触发条件**：
- 新监管政策涉及跨境资金或灰色通道时自动激活
- 或用户手动启动「资金回流验证」工作流时执行

## 策略二：堵疏不同步风险监控

**策略名称**：堵疏节奏错配监控

**具体内容**：

1. **量化"堵"的力度**：
   - 监控：灰色渠道清理进度（涉及案件数量、冻结资金量、被查机构数）
   - 指标：高频来自媒体报道 + 低频来自官方通报

2. **量化"疏"的进展**：
   - 监控：QDII 额度发放、跨境理财通升级进展、新合规产品备案数
   - 指标：月环比增速、与监管时间表对比完成度

3. **计算错配指标**：
   - 错配值 = f(堵的力度) − g(疏的进度)
   - 正向：资金围栏效应加剧，回流但干涸在岸
   - 大幅正向 → 预警：流动性陷阱微观版本风险
   - 债市久期错配：大量资金涌入短久期债券 → 久期压缩 → 后续期限溢价暴涨风险

4. **配套预警输出**：
   - 当错配值超过阈值时，输出各市场的资金压力热力图（A 股/债市/另类资产）
   - 提示用户策略优先级：在"疏"实质落地前，不激进加仓

**应用场景**：墨枢的「风险管理模块·尾部风险监控」
- 在风控仪表盘中作为一个独立的政策节奏监控指标
- 与全市场资金流向数据联动

**触发条件**：
- 每次有重大整治政策出台后，自动进入监控周期
- 每周自动更新错配值
- 错配值连续 3 周上升且超过设定阈值 → 主动推送预警

## 策略三：合规渠道受益方识别与渐进配置

**策略名称**：合规渠道受益链识别

**具体内容**：

1. **受益链条分析**：
   - **一级传导**：头部券商（灰色资金回流 + 合规产品需求上升 → 经纪/资管收入增长）
   - **二级传导**：低估值蓝筹（散户防御性配置 3-6 月 + 机构流动性改善增配 3-6 月）
   - **三级传导**：指数 ETF 规模扩张（资金通过合规渠道进入，被动产品首当其冲）

2. **渐进型建仓策略**：
   - **不一次性打满仓位**：等待明确的催化剂信号（例如 QDII 额度实际扩大/券商月报超预期/某头部券商保荐承销环比回升）
   - **聚焦头部 3-5 家券商**：只有头部才能承接合规资金回流
   - **LDV 蓝筹分类配置**：区分 LDV-高股息（防御）/ LDV-慢增长（持有）/ LDV-超跌（反弹布局）

3. **催化剂信号库**：
   - QDII 新额度公告 / 跨境理财通扩容公告
   - 头部券商月报环比超 +10%
   - 沪港通或深港通南向资金连续净流入 5 日
   - AH 溢价指数收窄至 130 以下

4. **阈值再平衡**：
   - 当某标的涨幅超 15%（当前已有影子）→ 部分止盈
   - 等待回调或下一个催化剂再补仓

**应用场景**：墨枢的「策略研究模块·行业配置子模块」
- 在输出行业/板块推荐时，结合当前的合规渠道受益判断
- 在仓位管理环节，作为头寸建立的节奏参考
- 在信号监控环节，使用催化剂信号库触发操作建议

**触发条件**：
- 当前场景中已有重大整治+合规化的政策趋势
- 用户进入行业配置或组合再平衡流程时自动加载受益链分析
- 当催化剂信号库中任一信号触发时，输出操作提醒
- 当某标的涨幅碰触阈值，输出止盈/等待建议
