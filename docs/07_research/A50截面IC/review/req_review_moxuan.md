### ⚠️ WARN — 需T+7前补充

**WARN-1**: §3.1.3 复权一致性校验公式 `adj_factor[t]/adj_factor[t-1] ≈ close[t]/pre_close[t]` 需确认技术正确性。pre_close 在 tushare 中已是前日收盘价（非原始值），推导中 adj_factor 约分后恒等——该式实际校验 adj_factor 是否 ≈ 1。建议与玄知确认后修正。

**WARN-2**: §3.3 `a50_universe` 未定义索引。该表按 `ts_code + in_date` 查询频繁，无索引会导致全表扫描。建议补充：
```sql
CREATE INDEX idx_universe_code ON a50_universe(ts_code);
CREATE INDEX idx_universe_in_date ON a50_universe(in_date);
```

**WARN-3**: 停牌日因子计算行为未定义。当某成分股连续停牌导致因子回看窗口不足（如停牌超20日），`momentum_20d` 应返回 NULL 还是使用截至可获取的最大窗口？建议在 §4.1 补充停牌缺失值填充规则。

### 📝 建议（非阻塞）

- §4.1 `roe_ttm` / `profit_margin` 依赖财报数据，应标注数据源依赖状态，避免 T+14 发现不可算
- §3.2.1 `num_stocks` 极端情况：若全部50只成分股均停牌，该字段值为0，建议定义下限阈值（如 < 30 则对应截面IC标记为无效）
