# S001 方向准确率问题 — 征询Claude意见

## 背景

S001（资金回流情景推演）是"八部门整治A股影响"策略体系中的一个模块。它通过四重折扣因子（政策α₁、市场α₂、流动性α₃、合规α₄）的乘积，计算有效入市资本，然后使用蒙特卡洛模拟（GBM路径模拟）输出三种情景：乐观/基准/悲观。

目标：对A股核心标的，预测未来60天的价格方向（UP/FLAT/DOWN）。

## 当前算法

```
有效因子积 = α₁^w₁ × α₂^w₂ × α₃^w₃ × α₄^w₄

GBM漂移 = α×0.08 + (1-α)×(-0.03)    （α = 有效因子积）
         ↑          ↑
      上涨年化   下跌年化

60天后判断方向：净收益 > 2% → UP, < -2% → DOWN, 之间 → FLAT
```

其中α₁~α₄范围[0,1]：值越高代表越有利资金回流。因子的设计初衷是**调低α值来抑制预期收益**——如果政策打压（α₁低），期望收益下降甚至为负，系统输出DOWN。

## 问题

12个月滚动回测（以中国石油601857为例），方向准确率只有**8.3%~41.7%（取决于参数设置）**，远低于65%的目标。

## 已尝试的修复

### Plan A（调参）
1. α₁动态联动S002同步性指数：sync_index<0.5 → α₁降[0.3,0.5]
2. 动量修正：20日收益率>5%→情景正向偏移
3. 多窗口测试：1/2/3个月分别测试
4. 权重网格搜索：w₁~w₄ ∈ [0.5,2.0]步长0.25

结果：8.3% → 41.7%（2个月窗口最佳），触达天花板。

### Plan B（结构改进）
B1：UP/DOWN阈值从2%降至1.5%
B2：放宽漂移公式至 α×0.15+(1-α)×(-0.08)
B3：3配置集成投票（保守/标准/激进三路取2）

结果：模拟数据下仍约41.7%。**根因**：601857的因子模拟值始终高于漂移中性点~0.35，所有配置一致预测UP，集成投票全年无分歧。

## 根因诊断

### 数据限制（可能是问题的核心）
系统通过akshare获取A股实盘数据，但当前环境运行在代理（v2rayN）下，akshare/yfinance均无法联外。回测目前使用校准模拟数据，模拟数据的因子值分布可能偏离真实市场——真实市场中政策收紧期α₁会降至0.3以下，但模拟数据中从未出现。

### 模型结构瓶颈
即便用模拟数据，当前GBM框架的天花板约45%：
- 四因子等权乘积 → 因子值趋中（0.5~0.9）
- GBM漂移公式设计为"调低因子→抑制收益"，而非"调高因子→精确预测涨幅"
- 60天窗口可能不足以捕捉政策传导效应
