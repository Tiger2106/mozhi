
### 数据解读
- **仅28.64%的交易日全体50只股票完整**。绝大多数交易日（71.36%）缺少部分成分股。这是正常的，原因如下：
  - A50成分股有上市时间差（部分股票2017年/2018年才上市）
  - A50指数成分股会定期调整（调入调出），但本数据库按当前A50成分股列表存储
- **近期（2025年8月及2026年1月）连续多天只有49只**：可能是某只股票退市或长期停牌。
- **无重复记录**：数据唯一性校验通过。

### 对Schema设计的直接影响
1. **截面IC需要处理非齐次面板**：每个时间截面的样本量不同（23~50只）。IC = corr(factor_value, forward_return)，需要动态调整自由度。
2. **最小截面样本量**：需设定最低股票数阈值（建议≥30只），低于此阈值的日期从回测中剔除。
3. **2026年近期缺失模式**：近期交易日连续缺1只，可能涉及成分股调整。建议核实当前A50成分股名单是否与数据库一致。
4. **建议采用动态截面法**：每期使用该期实际存在的股票计算IC，而非固定50只。

## 7. 结论与建议

### 数据质量总评
```
✅ OHLCV核心数据: 优 (0缺失，19年覆盖)
✅ adj_factor: 优 (0缺失，突变合理)
✅ pre_close: 优 (仅0.0145%偏差，均为合法市场事件)
⚠️ 时间戳对齐: 中 (仅28.64%交易日全50只对齐)
⚠️ free_float_share: 差 (62.62%缺失)
❌ turnover_rate: 完全缺失
❌ total_mv/circ_mv: 完全缺失
❌ 外部因子库: 仅1只股票有因子数据
```

### 对A50截面IC计算的直接影响

| 维度 | 可行性 | 注意事项 |
|------|--------|---------|
| 收益率计算 | ✅ | 使用 `close * adj_factor` 后复权，计算 forward_return |
| 动量因子 | ✅ | 从 `close * adj_factor` 自行计算 N日收益率 |
| 反转因子 | ✅ | 同上 |
| 波动率因子 | ✅ | 从 `close`/`pct_chg` 计算 N日滚动标准差 |
| 流动性因子 | ⚠️ | `volume` 可用，`amount` 可用；但 `turnover_rate` 缺失，需自行计算 |
| 市值因子 | ❌ | `total_mv`/`circ_mv` 缺失，需从其他数据源补全 |
| 估值因子 | ⚠️ | `pe`/`pe_ttm`/`pb` 字段存在但需验证数据质量 |
| 自由流通调整 | ❌ | `free_float_share` 49/50只股票缺失率约65%，几乎不可用。基于free_float的因子设计应直接排除。 |

### 建议
1. **IC计算Pipeline从stock_daily基础字段衍生因子**：不依赖已有因子库。
2. **回测起点建议2014年后**：降低非齐次面板的影响。
3. **最小截面≥30只**：剔除样本量不足的日期。
4. **需要task_id额外数据补全**：自由流通市值、因子库数据等可后续通过tushare API补充。
5. **建议新增索引**：当前主键已涵盖 `(ts_code, trade_date)` 复合索引，建议补充 `(trade_date)` 独立索引以提高按日期截面的查询性能。
